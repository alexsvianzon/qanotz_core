# QANotz Core

This is the core functionality for the QANotz app and CLI tool. You can use this to build your own QANotz-like applications.

## Key Features

 - **Database**: The simple lookup style database for accessing all files.
 - **Parser**: Parse QANLang and format it for viewing
 - **File IO + OS Operations**: Simple file classes for reading and writing files, as well as OS operations like accessing folders and getting the date.
